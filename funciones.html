Funciones

¿Qué es?

Las funciones son bloques o segmentos de código que reciben un nombre y los cuales realizan o ejecutan una tarea especifica dentro de un programa, esta herramienta es útil y utilizada dentro del paradigma de la programación estructurada, ya que permite dividir un programa de gran complejidad en segmentos de código más pequeños donde cada uno cumple una tarea diferente y este se puede repetir una cantidad N de veces dentro de un programa dependiendo de cuál sea la necesidad. Dichas funciones tienen algunas reglas a tener en cuenta como parámetros de entrada, su salida, nombre y demás cosas.

Además, se sabe que existen varios tipos de funciones las cuales tienen una utilidad diferente y es importante conocer cada una de ellas con el fin de saber en qué momento y para que usarlas.

Sintaxis

Para la programación de funciones de Matlab basta con crear un archivo o hoja en blanco inicialmente, se procede a crear la función a partir de escribir el comando function una vez se tiene esto se asignan las salidas de la función se iguala y se digita el nombre del documento en donde se desea emplear la función y sus respectivas entradas, como es habitual un end para terminar la operación.
