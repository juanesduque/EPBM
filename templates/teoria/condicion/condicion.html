Estructuras de condición

¿Qué es?

Los condicionales son estructuras que permiten comprobar expresiones lógicas, es decir, permiten evaluar si el resultado de dicha expresión es falsa o verdadera y a partir de este resultado, tomar una decisión de si ir por un camino o por otro.

De esta manera, esta estructura permite tener un control de flujo de ejecución del programa, lo que hace de esta una herramienta muy potente, pues permite que un programa sea más dinámico u complejo. Dentro de esta estructura se encuentran tres tipos, la condición if, else if y else; cada una de estas cumplen una función dentro del control que estos brindan. Por ejemplo, el if evalúa directamente si la expresión se cumple, a partir de esta decisión toma un camino u otro, posteriormente se tiene el else if, si en el if el resultado fue negativo, la condición else if permite evaluar una nueva expresión y comprobar si esta si se cumple, de igual manera, dependiendo del resultado tomará un camino u otro. Por último, se tiene el else, el cual brinda la posibilidad de que, si no se cumplió ninguno de los anteriores, se ejecutara por ese camino.
