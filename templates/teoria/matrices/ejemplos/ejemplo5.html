Ejemplo #5


Enunciado


Se requiere desarrollar un programa en el que se ingrese por teclado una matriz de enteros y calcule el número mayor y el número menor de estos números ingresados dentro de la matriz.


Solución

                
     1 filas = input('Ingrese el número de filas: ');
     2 columnas = input('Ingrese el número de columnas: ');
     3 A = zeros(filas);
     4  mayor = 0;
     5 menor = 10000;
     6 for i = 1:filas
     7     for j = 1:columnas
     8         A(i,j) = input('Ingrese un número: ');
     9          if A(i,j) > mayor
    10             mayor = A(i,j);
    11         end
    12         if A(i,j) < menor
    13              menor = A(i,j);
    14         end
    15     end
    16 end
    17 disp(mayor)
    18 disp(menor)

Explicación

                
     1 Se declara una variable en la cual se guarda el número de filas ingresadas por teclado por el usuario.
     2 Se declara una variable en la cual se guarda el número de columnas ingresadas por teclado por el usuario.
     3 Se declara una matriz de nombre A y se inicializa llenándola de ceros.
     4 Se declara una variable llamada mayor y se iguala a cero para inicializarla.
     5 Se declara una variable llamada menor y se iguala a un valro muy grande para que luego haga la comparación y de este modo inicializarla.
     6 Se inicia el primer ciclo for con el indexador i el cual va a recorrer las filas.
     7 Se inicia el segundo ciclo for con el indexador j el cual va a recorrer las columnas.
     8 Se pone el nombre de la matriz junto con sus indexadores correspondientes y se iguala al comando que permite que el usuario ingrese el valor.
     9 Se pone la sentencia if para evaluar cuál número es mayor.
    10 Si el número ingresado es mayor, se igual la variable mayor a este valor de la matriz.
    11 Se cierra la sentencia if que hace la evaluación de qué número es mayor.
    12 Se pone la sentencia if para evaluar cuál número es menor.
    13 Si el número ingresado es menor, se igual la variable menor a este valor de la matriz.
    14 Se cierra la sentencia if que hace la evaluación de qué número es menor.
    15 Ahora se cierra el ciclo for de las columnas.
    16 Por último, se cierra el ciclo for de las filas.
    17 Además, se pone el comando disp con la variable que contiene el número mayor para que este la muestre.
    18 También, se pone el comando disp con la variable que contiene el número menor para que este la muestre.
