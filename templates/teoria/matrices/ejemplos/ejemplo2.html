Ejemplo #2


Enunciado


Se requiere desarrollar un programa que llene una matriz de tamaño 3x3 con números ingresados por teclado y posteriormente, imprimir esta matriz en la pantalla.


Solución

                
    1 f = 'Ingrese un número: ';
    2 A = zeros(3);
    3 for i = 1:3
    4     for j = 1:3
    5          A(i,j) = input(f);
    6     end
    7  end

Explicación

                
    1 Se declara un variable la cual contiene la frase que se va a mostrar para interactuar más con el usuario.
    2 Se declara una matriz de nombre A y se inicializa llenándola de ceros.
    3 Se inicia el primer ciclo for con el indexador i el cual va a recorrer las filas.
    4 Se inicia el segundo ciclo for con el indexador j el cual va a recorrer las columnas.
    5 Se pone el nombre de la matriz junto con sus indexadores correspondientes y se iguala al comando que permite que el usuario ingrese el valor.
    6 Ahora se cierra el ciclo for de las columnas.
    7 Por último, se cierra el ciclo for de las filas.
