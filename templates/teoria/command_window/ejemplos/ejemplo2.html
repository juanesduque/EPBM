Ejemplo #2


Enunciado


Se requiere desarrollar un programa que almacene en dos variables, a y b, dos valores diferentes, 5 y 4, ya que posteriormente van a ser usados en una función que va a realizar diferentes operaciones con ellos.


Solución

                
    1   a = 5;
    2   b = 4;

Explicación

                
     1 Se escribe el nombre de la variable y se iguala al valor deseado en el command window, ya que de esta manera lo que hace el software es almacenar la variable con dicho valor.
     2 Se escribe el nombre de la otra variable y se iguala al valor deseado en el command window, ya que de esta manera lo que hace el software es almacenar la variable con dicho valor.
