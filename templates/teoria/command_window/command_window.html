Command window

¿Qué es?

En esta ventana se ejecutan de forma interactiva e instantánea las instrucciones de Matlab o las operaciones que se deseen realizar rápidamente, además, en esta ventana se muestran los resultados correspondientes si es el caso. Esta es una de las ventanas más importantes ya que existe desde las primeras versiones de Matlab.

Sintaxis

El command window se encuentra ubicado en la parte inferior de la pantalla, en esta ventana se pueden digitar diversos comandos para que el usuario tenga una experiencia optima. Algunos de estos comandos son Help, el cual al digitarse en el command window, dirige al usuario a un directorio con información acerca de todas las funciones y comandos de Matlab. Otra de las funciones que se pueden usar en el command window es auto completar el nombre de una función oprimiendo la tecla tab facilitando el trabajo y optimizando el tiempo. Una de las tares más importantes del command window es mostrarle al usuario los errores y la línea en la cual se encuentra el error después de compilar un fichero *.m. Para encontrar este error fácilmente este se encuentra subrayado con un enlace que lleva a la línea específica del error después de dar click sobre dicho enlace.

Ejemplos

En la sección de ejemplos se pretende poner en práctica el concepto que fue explicado anteriormente con el fin de interiorizar el tema, verlo en ejecución, ver cómo se utiliza y para qué, ya que de esta manera es que se logra un aprendizaje significativo. En los ejemplos se encontraran ejercicios que pretenden dar solución a ciertos problemas planteados, esto con el fin de llevar la explicación más a la vida real y de qué mejor manera que poniendo porblemas que se presentan en la vida cotidiana y resolviéndolos con este lenguaje de programación.
